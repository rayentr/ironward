MCP server · CLI · VS Code extension

Security scanning for the vibe coding era.

Ironward is an open-source Model Context Protocol server and CLI that finds hardcoded secrets, auth bugs, SQL injection, XSS, IDOR, and vulnerable dependencies — and opens a fix PR for any finding. Runs inside Cursor, Claude Code, and VS Code. Four of its tools work fully offline — no API key required.

Install in 3 lines View on GitHub → 
// ~/.cursor/mcp.json
{
  "mcpServers": {
    "ironward": {
      "command": "npx",
      "args": ["-y", "ironward@latest"],
      "env": { "ANTHROPIC_API_KEY": "sk-ant-..." }
    }
  }
}
# one shot
claude mcp add ironward -- npx -y ironward@latest
// .vscode/mcp.json
{
  "servers": {
    "ironward": {

# scan without any IDE setup — no API key needed
npx ironward scan ./src
npx ironward scan-secrets ./src
npx ironward scan-code ./src
npx ironward scan-deps package.json
npx ironward scan-url https://example.com
ANTHROPIC_API_KEY is only needed for AI-backed tools (scan_auth_logic, scan_sqli, scan_xss, scan_idor, fix_and_pr). Run ironward free to see what works offline.

Nine tools, wired into your editor.

Each tool is independently useful. Your IDE chooses when to call which — or you can invoke them directly from the CLI.

scan_for_secrets
665 pattern families, zero network.

AWS, GCP, Azure, Stripe, PayPal, GitHub, OpenAI, Anthropic, Slack, Postgres/Mongo URLs, private keys, Ethereum/Solana wallets, Supabase/PlanetScale/Turso + Shannon-entropy fallback. Placeholder + UUID/SHA allowlist keeps false positives near zero.

offline · regex + entropy
scan_code
27 static rules, pure offline.

eval, new Function, command injection, path traversal, weak crypto (MD5/SHA-1/DES), SSRF, open redirects, prototype pollution, CORS wildcard, JWT alg:none, logged secrets, debugger left in code, and more.

offline · static analysis
scan_auth_logic
Reasoning models catch what patterns miss.

Backwards auth checks, missing ownership checks, privilege escalation, bypassable middleware, JWT alg:none acceptance, session fixation. A keyword pre-filter skips the model on unrelated files.

AI · keyword pre-filter
scan_sqli
Injection across six languages.

String concat, template literals, f-strings, %-format, str.format, Sprintf, ORM raw / $queryRawUnsafe / knex.raw. 24-rule cross-language pre-filter, model confirms.

AI · multi-language pre-filter
scan_xss
DOM + server-side XSS, all frameworks.

innerHTML, dangerouslySetInnerHTML, Vue v-html, Angular bypassSecurityTrust*, Svelte {@html}, EJS/Mustache unescaped output, eval, reflected Express/Koa responses.

AI · sanitizer-aware
scan_idor
Missing ownership checks on every route.

Flags routes that read an ID from params/body/query and hit the DB without an owner check. Detects raw prisma.update({ data: req.body }) mass-assignment. Counts owner hints to avoid false positives.

AI · owner-hint pre-filter
scan_deps
Every dependency, every CVE + supply-chain intel.

Parses package.json, requirements.txt, Pipfile.lock and queries OSV.dev. Also detects typosquats, abandoned packages, known malware, and copyleft/unlicensed deps. Sorted by CVSS.

offline · OSV.dev · typosquat · malware
scan_url
Letter-graded web scan, no install.

Probes security headers, cookies, CORS, exposed .env / .git, JS source maps, admin panels, public API docs, embedded Supabase/Firebase keys, TLS cert expiry, error leakage. Returns an A+/A/B/C/D/F grade.

offline · HTTP probes
fix_and_pr
Zero to merged — one tool call.

Hands any finding to a reasoning model for a surgical patch, creates a branch, commits across multiple files, validates the fix against the same scanner, opens a PR with OWASP references. dryRun: true to preview.

AI · GitHub REST · multi-file + validation

What makes it different.

01
Offline-first.

Four of nine tools — secrets, static code, deps, URL — run fully offline with no API key. Bring a key only when you want AI reasoning for auth/SQLi/XSS/IDOR or automated fixes.

02
It fixes the bug, not just finds it.

fix_and_pr turns a finding into a reviewable pull request. Multi-file fixes with validation loop — the patch is re-scanned before the PR opens.

03
Bring your own model.

Works with Anthropic, OpenAI, Gemini, Groq, or a fully-local Ollama install. Your key stays local. No Ironward cloud, no per-seat pricing.

04
Three-line install.

Paste a config, restart your editor, done. Or run npx ironward scan ./src with no setup whatsoever.

Try it without installing anything.

Every scanner has a direct CLI mode. Drop it on a directory, get output in your terminal.

# one command, runs everything offline
npx ironward scan ./src

# individual scans
npx ironward scan-secrets ./src
npx ironward scan-code ./src
npx ironward scan-deps package.json requirements.txt
npx ironward scan-url https://example.com

# bring your own model for AI-backed scans
npx ironward login

# start the MCP server manually (what your IDE does)
npx ironward serve
